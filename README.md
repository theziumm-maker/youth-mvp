# Youth MVP — 데일리 페이스 기록

하루 3초, 매일 같은 조건으로 얼굴을 기록하고 변화를 추적하는 앱.

- 촬영: 실시간 품질 게이트(밝기·선명도) + 3초 카운트다운 촬영
- 분석: 픽셀 기반 근사 지표(종합/피부톤/질감/밝기/선명도) 추세 그래프
- 전후: 두 날짜 비교 슬라이더 + 지표 델타

## 배포

**DEPLOY.md**를 보세요 — GitHub + Render 무료 배포를 클릭 단위로 안내합니다.

## 로컬 실행

```
npm install
npm start
```
DATABASE_URL 없이 실행하면 메모리 모드(테스트용)로 동작합니다.
